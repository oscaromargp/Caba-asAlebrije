## 📋 Descripción

**Cabañas Alebrije** es el sitio web oficial de un alojamiento turístico único ubicado en el corazón de la Reserva de la Biosfera Ría Celestún, Yucatán, México.

El proyecto consiste en una landing page premium de modo oscuro (dark mode) diseñada para comunicar la propuesta de valor del negocio: desconexión, ecoturismo y hospitalidad orgánica. El sitio incluye:

- 🌐 **Traductor automático** ES ↔ EN
- 📱 **Diseño 100% responsive**
- ✨ **Animaciones y efectos visuales**
- 📋 **Formulario de reservas** con integración WhatsApp
- 🖼️ **Galería interactiva**
- 💬 **Sección de FAQ**

## ⚙️ Configuración

### Cómo Clonar el Repositorio

```bash
# Clonar repositorio
git clone https://github.com/oscardeveloper/alebrije-cabanas.git

# Entrar al directorio
cd alebrije-cabanas

# Abrir en navegador
open index.html
# o en Windows
start index.html
```

### Variables Editables

En `styles.css`:

```css
/* Color primario - cambiar aquí */
--primary-500: #06B6D4;

/* Fondos */
--bg-primary: #000000;
--bg-secondary: #09090B;

/* Textos */
--text-primary: #FFFFFF;
```

### Personalización de Contenido

El contenido se encuentra en `index.html` en atributos `data-es` (español) y `data-en` (inglés).

**Para cambiar textos:**
1. Busca el elemento en HTML
2. Modifica el atributo `data-es` para español
3. Modifica el atributo `data-en` para inglés

## 🚀 Deployment

### Netlify (Recomendado)

1. Ve a [Netlify](https://netlify.com)

3. Deploy automático

## 🎨 Personalización

### Colores

El color principal es turquesa (#06B6D4). Para cambiar:

1. **CSS:** Edita `--primary-500` en `styles.css`
2. **HTML:** Actualiza los gradientes en el hero

### Testimonios

Añade nuevos testimonios en la sección `#testimonios`:

```html
<div class="testimonial-card glass-card">
    <div class="testimonial-stars">★★★★★</div>
    <p class="testimonial-text">"Tu testimonio aquí"</p>
    <div class="testimonial-author">
        <div class="author-avatar">X</div>
        <div class="author-info">
            <span class="author-name">Nombre</span>
            <span class="author-location">Ciudad</span>
        </div>
    </div>
</div>
```

### FAQ

Añade nuevas preguntas en `#faq`:

```html
<div class="faq-item glass-card">
    <button class="faq-question" onclick="toggleFaq(this)">
        <span data-es="Tu pregunta en ES" data-en="Tu pregunta en EN">Tu pregunta</span>
        <span class="faq-icon">+</span>
    </button>
    <div class="faq-answer">
        <p data-es="Tu respuesta en ES" data-en="Tu respuesta en EN">Tu respuesta</p>
    </div>
</div>
```
